# Masterfilter - Web Scraping Platform

Sistema de web scraping empresarial desarrollado en Python/Django para extracción y análisis de datos.

## Aplicaciones

- **masterfilter**: Aplicación principal de scraping
- **movil**: Módulo de scraping de datos móviles
- **whatsapp**: Integración con WhatsApp

## Tecnologías

- Python 3.x
- Django
- PostgreSQL/MySQL
- Gunicorn/Hypercorn
- Node.js (frontend tooling)

## Instalación
```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

## Configuración

Crear archivo `.env` con las variables necesarias.
